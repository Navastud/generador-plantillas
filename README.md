# Generador Plantillas

Aplicación para la generación de plantillas de manera dinamica, basandose en una plantilla origen en formato excel para extraer de datos de las columnas y filas correspondientes mediante una configuración inicial para posteriormente insertar los valores en una plantilla destino aplicando la misma lógica. 


## Tecnologías Utilizadas

* Java 8
* Java Swing
* Maven
* Git-Flow
* Apache POI

## Ejecución

Ejecutar el archivo generador-plantillas.jar
